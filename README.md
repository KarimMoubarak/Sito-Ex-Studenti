# 🌐 Sito Web Associazione – Progetto Scolastico

Questo progetto è un sito web realizzato per un’attività scolastica, con l’obiettivo di creare una piattaforma informativa e funzionale per un’associazione.

Il sito è stato sviluppato andando oltre le richieste iniziali, aggiungendo miglioramenti grafici, funzionali e strutturali per renderlo più moderno e completo.

---

## 📌 Obiettivi del progetto

Le richieste iniziali prevedevano la realizzazione di un sito con:

- 🔗 Collegamento dalla homepage del sito della scuola
- 🏠 Pagina di ingresso con accesso alle sezioni principali
- 📄 Pagina per il download dei documenti importanti
- 📞 Pagina contatti (email e telefono dell’associazione)
- 🏛️ Pagina del consiglio direttivo in carica fino al 2026
- 🔗 Pagina con link a siti di interesse
- 💳 Pagina per iscrizione o donazione con modulo e pagamento tramite bonifico bancario

---

## 🚀 Implementazioni aggiuntive

Durante lo sviluppo sono state aggiunte funzionalità extra non richieste inizialmente:

- 🎨 Miglioramento grafico dell’interfaccia utente
- 🌗 Sistema di temi dinamici (stagionali e automatici)
- 📱 Maggiore attenzione alla responsività (mobile/tablet)
- 🧭 Navigazione più fluida e intuitiva tra le pagine
- 🧩 Struttura del sito più modulare e scalabile

---

## 🧱 Struttura del sito

Il sito è organizzato nelle seguenti sezioni principali:

- 🏠 Home / Pagina di ingresso
- 📄 Documenti
- 📞 Contatti
- 🏛️ Consiglio direttivo
- 🔗 Link utili
- 💳 Iscrizione / Donazioni

---

## 🛠 Tecnologie utilizzate

- HTML5
- CSS3
- JavaScript Vanilla
- Sistema di temi basato su `data-theme`

---

## 🎨 Caratteristiche grafiche

- Interfaccia personalizzabile tramite temi
- Design adattivo (responsive)
- Supporto modalità giorno/notte
- Esperienza utente migliorata rispetto alla versione base richiesta

Questo lavoro è stato sviluppato come estensione delle richieste iniziali, con l’obiettivo di:

- migliorare l’esperienza utente
